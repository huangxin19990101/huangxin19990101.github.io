命令的格式

命令的说明

ls
参数 含义
-a 以.开头的文件为隐藏文件 , 需要-a才能显示 , .表示当前目录, ..表示上一级目录
-l 列表方式显示文件详细信息
-h 文件大小以 k 或者 m 为单位
通配符 含义
* 代表任意个数个字符 ls a*a
? 代表任意一个字符，至少 1 个 ls a?a
[] 表示可以匹配字符组中的任一一个 ls a[123]a
[abc] 匹配 a、b、c 中的任意一个
[a-f] 匹配从 a 到 f 范围内的的任意一个字符
cd
mkdir

cp

mv

可以移动目录 , 文件 或者 重命名

cat

查看文件内容

-b非空行显示行号, -n所有行显示行号

more

和cat类似, 但是会从头开始查看 , cat是直接输出到结尾

grep
选项 含义
-n 显示匹配及行号
-v 显示不匹配的所有行
-i 忽略大小写
echo

在控制台打印参数指定的文字

echo hello

控制台输出 : hello

重定向 > >>

通常配合echo 使用

>表示覆盖文件

>>表示追加到文件结尾

管道 |
scp远程复制

which

bin和sbin

用户权限

序号 权限 英文 缩写 数字
1 读 read r 4
2 写 write w 2
3 执行 execute x 1
chmod

修改文件/目录权限

如果去掉目录的x权限则无法访问

chown
组管理
用户管理

查看用户信息

usermod添加附加组

系统信息

date 和 cal查询时间和日期
df-h显示磁盘剩余空间
du-h[目录名] 显示目录下文件大小
进程
ps命令
top

动态显示运行中的进程并排序

kill

find

软链接
压缩
tar
选项 含义
c 生成档案文件, 创建打包文件
x 解开档案文件
v 列出进度
f 指定档案文件名称 f后面一定是.tar文件, 必须放最后
gzip
